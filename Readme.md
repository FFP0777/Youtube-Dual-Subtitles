# YouTube 雙字幕即時翻譯工具 (Tampermonkey 腳本)

基於 Tampermonkey 的油猴腳本，可以在觀看 YouTube 影片時，  
即時翻譯字幕並顯示**雙語字幕**，支援：
- 可調整字體大小、字體色彩
- 移動與縮放字幕框，隨意擺放位置

適合學習英文、韓文、日文等外語，或提升影片觀看體驗！
